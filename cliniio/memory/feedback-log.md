Record of founder feedback on agent outputs. Used to calibrate agent behaviour over time.

## Format

Each entry follows this structure:

```

**What was produced:** Brief description of the output
**Founder feedback:** What was good, what needed changing, what was wrong
**Action taken:** How the agent or rules were adjusted
```
